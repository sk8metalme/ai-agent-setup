# コーヒー豆紹介の出力フォーマット

## 概要

Step 11-2 で使用する「今日のコーヒー豆」の出力フォーマット定義です。
[coffee_beans.md](coffee_beans.md) からランダムに1つ選んで紹介してください。

## 出力テンプレート

```markdown
---

☕ **本日のコーヒー: {コーヒー豆名}**

| 項目 | 評価 |
|------|------|
| 苦さ | {★で表現（1-5）} |
| 酸味 | {★で表現（1-5）} |
| 香り | {★で表現（1-5）} |
| コク | {★で表現（1-5）} |
| 甘み | {★で表現（1-5）} |

**特徴**: {フレーバープロファイル}

**おすすめの飲み方**: {抽出方法}

**ひとこと**: {豆の背景・産地情報・おすすめポイント}

---

また明日も、新しい知見と美味しいコーヒーをお楽しみに！☕
```

## 実装例

**エチオピア イルガチェフェ**:

```markdown
---

☕ **本日のコーヒー: エチオピア イルガチェフェ**

| 項目 | 評価 |
|------|------|
| 苦さ | ★★☆☆☆ |
| 酸味 | ★★★★☆ |
| 香り | ★★★★★ |
| コク | ★★★☆☆ |
| 甘み | ★★★★☆ |

**特徴**: フローラル、フルーティー、紅茶のような上品さ

**おすすめの飲み方**: ハンドドリップ

**ひとこと**: コーヒー発祥の地エチオピアが誇る最高級品種。標高2000m以上の高地で栽培され、花のような華やかな香りが特徴です。

---

また明日も、新しい知見と美味しいコーヒーをお楽しみに！☕
```

## 選択方法

- `coffee_beans.md` から**ランダムに1つ**選択
- 毎回同じにならないように工夫（日付ベースのシードなど）
- すべての豆が均等に選ばれるようにする

## 注意事項

- **完全自律実行**: ユーザーへの質問禁止
- **カスタマイズ歓迎**: ひとことコメントはアレンジ可能
- **敬称**: 豆名は正式名称を使用（例: "コロンビア スプレモ"）
