## プロジェクト概要
**名称**: ai-agent-setup  
**目的**: AIエージェントの設定ファイル類を配布し、即座に活用できるようにする

### 核心機能
1. **マルチAI対応**: Claude, Codex, Cursor
2. **ワンコマンドセットアップ**: upxを使用した簡単導入
3. **安全な導入**: 既存環境を破壊せず、元に戻せる仕組み
4. **日本語対応**: 日本市場に特化

### 設計原則
- シンプルであること
- 変化に柔軟であること  
- 導入が簡単であること
- 既存環境を破壊しないこと・元に戻せること
